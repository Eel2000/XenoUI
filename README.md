# 🚀 XenoUI

> A modern, cross-platform UI framework for .NET that brings elegant user interfaces to Android and iOS

## Overview

XenoUI is a next-generation UI framework built for the .NET ecosystem, designed to empower developers in creating beautiful, responsive applications that run seamlessly across Android and iOS platforms. With a focus on performance, developer experience, and rich visual capabilities, XenoUI provides a comprehensive toolkit for building modern mobile applications.

## ✨ Features

- **🎯 Cross-Platform Support** - Write once, deploy to both Android and iOS using .NET
- **🎨 Modern UI Components** - Pre-built, customizable components for rapid development
- **✨ Smooth Animations** - Built-in animation system for polished, fluid UX
- **⚡ High Performance** - Optimized rendering pipeline and efficient composition
- **🛠️ Developer Friendly** - Intuitive APIs and comprehensive tooling
- **📦 Modular Architecture** - Clean separation of concerns with core and platform-specific implementations

## 📋 Project Structure

```
XenoUI/
├── XenoUI.Core/              # Core framework and components
│   ├── Animations/           # Animation system
│   ├── Compositor/           # UI composition and rendering
│   ├── Doddb/               # Data management layer
│   └── Pillars/             # Architectural foundation
├── XenoUI.StandardLibrary/   # Standard UI components library
├── XenoUI.Android/          # Android platform implementation
├── XenoUI.Ios/              # iOS platform implementation
└── XenoUI.BuildTasks/       # Build system and tooling
```

## 🚀 Getting Started

### Prerequisites
- .NET 10.0 SDK or later
- Android SDK (for Android development)
- Xcode (for iOS development)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/XenoUI.git
cd XenoUI

# Build the solution
dotnet build
```
